{{$endraw}} desc: Here we first create a blog post and place the meta tag at the top where we add some properties to the post, like a post title, created date and author. We then fill out the rest of the post like a normal HTML page. Later in the index.html page we create a list of posts by using the each tag. Within that block we can refer to the properties that we created in the meta block of each file. {{$ endmeta }}
