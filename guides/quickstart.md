Welcome to Backlift. This guide will quickly walk you through the basics of creating a Backlift website, making some changes, adding dynamic data with template tags, and deploying your site to production. Let's get started!

## Creating a website

We'll start by creating a new website using the blank starter kit. Head over to [the create new website form](http://www.backlift.com/dashboard/createapp). You'll see a place to enter your new app's name. Pick something clever, but please only use alphanumeric characters. (no spaces) Leave the "blank" template selected and then click on the Create button.
